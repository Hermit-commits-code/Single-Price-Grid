## Overview

This project recreates the pricing card layout using semantic HTML and custom CSS variables.
It includes a desktop layout and a mobile breakpoint.

### The challenge

Users should be able to:

- View the optimal layout depending on their device screen size
- See hover and focus states for all interactive elements

### Built with

- CSS nesting
- Mobile responsive media query

### What I learned

- How fixed heights can cause overflow when stacked sections exceed the parent height
- Why `border-radius` on a parent needs clipping (`overflow: hidden`) when children have solid backgrounds
- How to switch a two-column grid to one column for mobile with a single media query override
